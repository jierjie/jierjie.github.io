shell的简介

概念

shell是什么？
shell 壳！

shell就是介于用户和操作系统内核之间的桥梁，让用户使用操作系统的时候更加方便！

分类

图形化shell： windows图形化界面
命令行shell: cmd bash（所有的linux操作系统都会自带这个shell） powershell

bash

linux系统下的自带的一个shell

我们可以在bash下执行所有的linux的命令，来实现平时通过图形化界面实现的操作！！

因为在想要执行linux命令，必须是在linux系统的bash环境下。

bash命令的格式

命令名称 -选项 -选项 参数 参数

ls

全称： list
功能： 展示当前文件夹下所有的文件内容

ls -a -a 是一个选项
功能： 显示当前文件夹下所有的内容 包括隐藏的文件

ls -l -l 是一个选项
功能： 将文件以列表的形式进行展示

如果有多个选项 可以依次将选项列在命令后面
ls -a -l

也可以将选项进行合并
ls -al

ls命令也可以有参数
ls 路径
功能 展示指定路径下的所有的文件信息

pwd

全称： print work directory

cd

全称： change directory 使用频率非常高！！

功能： 改变当前所处的工作目录

/ 当前电脑的根目录 里面包含c d 盘
~ 当前用户的根目录

touch

功能：创建文件

touch 文件名称 在当前的文件夹下创建文件

touch 文件的路径 在指定的文件夹下创建文件

mkdir

功能： 创建文件夹

mkdir 文件夹名称 在当前文件夹下创建新的文件夹

mkdir 文件夹路径 在指定的文件夹下创建新的文件夹

clear

功能： 清空屏幕

rm

全称： remove
功能： 删除指定的文件

rm 文件名称 删除当前目录下指定名称的文件

rm 文件路径 删除指定目录下的指定名称的文件

rm -r -r 选项指的是 递归的意思 递归删除！！

rm -f -f 选项是指 force 强制删除

注意： rm 和 rmdir删除的内容不进回收站！！！

rmdir

全称： remove directiory

rmdir 文件夹名称
rmdir 文件夹路径

rmdir只能够删除空文件夹，如果文件夹中有内容，则无法删除！！

cp

全称 copy

功能： 拷贝文件

cp 源文件(名称或者路径) 目标文件(名称或者路径)

cp -r 源文件夹 目标文件夹 可以用来复制文件夹！！

mv

全称 move
功能： 移动文件 类似于剪切

mv 源文件 目标文件

mv 移动文件夹不需要加任何选项，和文件操作一样！

mv也可以用来执行重命名的操作！

cat

功能：展示文件内容

cat 文件名/文件路径

cat是将所有的文件内容一次性的全部展示出来

less

功能：分页展示文件内容

less 文件名/文件路径

翻页： 使用空格 翻一行 方向键 回车键
退出浏览： q

tab键

智能补全

echo

echo用来打印内容

> 输出重定向

所有命令的输出默认都是在屏幕上
我们可以通过 输出重定向命令 将输结果 输出到指定的位置

>：　替换操作
>>: 追加操作
